# Product Search UI Lab

This static browser lab turns chapter 16's product-search case study into a visible UI.

It demonstrates:

- URL state for search, category, sort, and page
- filter changes resetting `page` to 1
- empty, loading, error, and ready states
- derived filter count and next-page state
- accessible labels and real buttons
- image dimensions to avoid layout shift
- optimistic favorite updates with rollback

## Run

Open `index.html` in a browser.

No install step or dev server is required.
